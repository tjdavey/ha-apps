This is a temporary copy of [bohdan-s's](https://github.com/bohdan-s) [SunGather Add-on](https://github.com/bohdan-s/hassio-repository) with support for Home Assistant's new 'App' format.

For documentation on SunGather see the [original repository](https://github.com/bohdan-s/hassio-repository). 

The new app format is required by Home Assistant Supervisor versions following April 2026 and the existing SunGather Add-on is no longer installable on those versions. This app is intended to be a stop-gap solution until the original add-on can be updated to support the new format, at which point this app will be removed from this repository.
